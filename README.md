# ConnectaTel - Telecom Data Analysis

## Objetivo del proyecto

Analizar el comportamiento de clientes de telecomunicaciones en Latinoamérica utilizando información de usuarios, planes y uso real del servicio.

El objetivo es identificar patrones de consumo, segmentar clientes, detectar outliers y generar recomendaciones comerciales para ConnectaTel.

---

## Datasets utilizados

- plans.csv
- users_latam.csv
- usage.csv

---

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

---

## Etapas del análisis

1. Carga y exploración de datos
2. Limpieza de datos
3. Detección de valores nulos y sentinels
4. Estandarización de fechas
5. Summary statistics
6. Visualización de distribuciones
7. Detección de outliers
8. Segmentación de clientes
9. Insights ejecutivos

---

## Cómo ejecutar el proyecto

1. Descargar el notebook `.ipynb`
2. Abrir en Jupyter Notebook o Google Colab
3. Cargar los datasets en la carpeta `/datasets`
4. Ejecutar las celdas en orden

---

## Resultados principales

- Identificación de segmentos de clientes por edad y uso
- Detección de usuarios intensivos
- Identificación de outliers
- Recomendaciones comerciales para optimizar planes y fidelización
